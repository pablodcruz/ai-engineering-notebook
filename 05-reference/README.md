## How To Use This Folder

- Link to a definition instead of redefining the same term inconsistently.
- Copy checklist questions into project reviews and customer discovery sessions.
- Record a resource only when it changes a build, explanation, or follow-up action.
- Verify time-sensitive API, model, pricing, product, and service information against current official documentation.

Reference notes summarize durable takeaways. They do not replace provider documentation or customer-specific policy.
